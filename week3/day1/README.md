# PySpark Learning Journey

## Day 1 - PySpark Basics

### PySpark
PySpark is the Python API for Apache Spark. It is used to process large amounts of data efficiently.

### SparkSession
SparkSession is the entry point to work with PySpark. It allows us to create DataFrames and execute Spark operations.

### DataFrame Creation
A DataFrame is a distributed table with rows and columns. It is the most commonly used data structure in PySpark.

### show()
The show() method displays the contents of a DataFrame in a tabular format.

### printSchema()
The printSchema() method displays the structure of a DataFrame, including column names and data types.

### select()
The select() method is used to retrieve specific columns from a DataFrame.

### Key Learning
Learned how to create a SparkSession, create a DataFrame, inspect its schema, and select required columns.
