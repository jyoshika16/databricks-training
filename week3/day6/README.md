## Day 6 - Filtering, Transformations and Data Type Operations

### filter()
Used to retrieve rows that satisfy a specified condition.

### where()
Alternative syntax for filtering records in a DataFrame.

### withColumn()
Creates a new column or modifies an existing column.

### cast()
Converts a column from one data type to another.

### sort()
Sorts records in ascending order by default.

### orderBy()
Sorts records in ascending or descending order using one or more columns.

### limit()
Returns only the specified number of rows.
