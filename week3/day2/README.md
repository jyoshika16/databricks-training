## Day 2 - DataFrame Operations

### select()
The select() method is used to retrieve specific columns from a DataFrame.

### alias()
The alias() method is used to provide a temporary name to a column in the output.

### withColumn()
The withColumn() method is used to create a new column or modify an existing column.

### withColumnRenamed()
The withColumnRenamed() method is used to permanently rename a DataFrame column.

### Key Learning
Learned how to select required columns, rename columns temporarily, create new columns, and rename existing columns.
