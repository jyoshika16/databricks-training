# REGEX Foundation Practice

In this practice session, I learned the basics of Regular Expressions (REGEX) and how pattern matching is used in SQL for extracting, validating and filtering text data.

I practiced different regex symbols and patterns using emails, phone numbers and mixed text values to understand how text extraction works in real-time data processing scenarios.

Topics Covered:
- [0-9] for matching digits
- [A-Z] and [a-z] for matching alphabets
- ^ for matching patterns at the beginning of text
- $ for matching patterns at the end of text
- + for matching continuous characters
- { } for exact number of characters
- REGEXP_SUBSTR for extracting matching values
- Email and phone number pattern extraction
- Domain and extension extraction using regex

Solved multiple practice queries related to emails, phone numbers and mixed text data to improve understanding of pattern matching and text validation concepts in SQL.
