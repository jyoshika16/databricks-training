# CASE Statements and Real-Time SQL Scenarios

In this practice session, I learned how CASE statements are used in SQL to apply conditions, categorize data and generate dynamic outputs based on business logic.

I practiced different real-time SQL scenarios using employee, order and sales data to understand how conditional logic works in reporting and data analysis.

Topics Covered:
- CASE WHEN conditions
- Conditional categorization
- Financial year logic
- Salary and bonus conditions
- Order and delivery status conditions
- Real-time reporting scenarios
- Date-based business conditions

Solved multiple practical queries to improve understanding of conditional expressions and business logic implementation in SQL.
