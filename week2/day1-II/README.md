# REGEX Assignment Practice

In this practice session, I solved different REGEX-based SQL queries to improve understanding of pattern matching, text extraction and data validation techniques.

I practiced extracting numbers, alphabets, domains, extensions and country codes from mixed text values, phone numbers and email addresses using REGEXP_SUBSTR.

Topics Covered:
- Extracting digits and alphabets
- Extracting text from beginning and end of strings
- Extracting exact number patterns
- Email username, domain and extension extraction
- Phone number and country code extraction
- Pattern matching using REGEX symbols and conditions

Solved multiple real-time text processing queries to improve understanding of regex-based data cleaning and validation in SQL.
