# Number Functions Practice

In this practice session, I learned how numeric functions are used in SQL to perform calculations, rounding operations and mathematical analysis on numeric data.

I practiced different number functions using employee salary and bonus data to understand how numerical processing works in real-world business applications.

Topics Covered:
- ABS to get absolute values
- ROUND to round numbers to specific decimal places
- CEIL and FLOOR for rounding up and down
- TRUNCATE to remove decimal values without rounding
- MOD to find remainder values
- POWER and POW for exponential calculations
- SQRT to find square roots
- SIGN to identify positive and negative values
- RAND to generate random numbers
- FORMAT to display formatted numeric values
- GREATEST and LEAST to compare multiple numeric values

Solved practice queries related to salaries, bonuses, taxes and employee experience data to improve understanding of numeric calculations and mathematical functions in SQL.
