# String Functions Practice

In this practice session, I learned how string functions are used in SQL to clean, format and manipulate text data stored in databases.

I practiced different string functions using employee names, emails, departments and city details to understand how text processing works in real-world applications.

Topics Covered:
- LENGTH and CHAR_LENGTH to count characters in strings
- UPPER and LOWER to convert text into uppercase and lowercase
- TRIM, LTRIM and RTRIM to remove unwanted spaces
- CONCAT and CONCAT_WS to combine multiple strings
- SUBSTRING, LEFT and RIGHT to extract specific parts of text
- INSTR and LOCATE to find positions of characters and words
- REPLACE to modify text values
- REVERSE to reverse strings
- LPAD and RPAD for string padding
- IFNULL and COALESCE for handling NULL text values

Solved multiple practice queries related to employee names, email addresses, departments and remarks to improve understanding of string manipulation and data formatting in SQL.
