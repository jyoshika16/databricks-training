# Window Functions Practice

In this practice session, I learned how window functions are used in SQL to assign rankings, row numbers and positions to records without grouping the entire table.

I practiced different window functions using employee salary, department and joining date data to understand how ranking and ordering operations work in real-time business scenarios.

Topics Covered:
- ROW_NUMBER() to assign unique row numbers
- RANK() to assign rankings with skipped positions for ties
- DENSE_RANK() to assign continuous rankings without skipping numbers
- PARTITION BY to divide data into groups
- ORDER BY inside window functions for sorting records

Solved practice queries related to employee salaries, departments, joining dates and order rankings to improve understanding of analytical and reporting functions in SQL.
